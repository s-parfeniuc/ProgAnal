# Exercises — index

All exercises from the course, **extracted from the slides** (lectures 03–23,
`CFA-for-FUN`, `CFA-for-Pi`) and the **exam exercise sheets**, grouped into three
macro-topics. Each entry has the statement first and an **Answer / Solution**
block wherever the slides or the solution sheet provide one; every item is tagged
with its source `[lec NN / slide M]` or `[sheet a/b]`.

| File | Macro-topic | Covers | Source lectures |

## Original sources in this folder
- `ProgramAnalysis_exercises_a.pdf` — exam sheet A (Ex. 1 logics, Ex. 2 AI, Ex. 3 CFA).
- `ProgramAnalysis_exercises_a_solutions.pdf` — full worked solutions for sheet A
  (embedded into the three files above).
- `ProgramAnalysis_exercises_b.pdf` — exam sheet B (Ex. 1 ISL, Ex. 2 GC, Ex. 3 CFA+DFA);
  no published solutions.

The exam-sheet exercises are cross-cutting, so each sheet's three questions are
placed in the matching macro-topic file (see the "Exam-sheet items" section at the
end of each): sheet A → X.1 (logics), X.2 (AI), X.3a (CFA); sheet B → ISL.3 / X.3b.

## How to use
Cover the **Answer / Solution** block and try each statement cold; the solutions
mirror the slide reasoning so you can check derivation shape, not just the result.
For deeper background on any logic/domain, follow the links into
[`../overviews/`](../overviews/README.md).

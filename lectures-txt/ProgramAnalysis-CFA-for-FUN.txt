Control Flow Analysis

• implicit in the program code and in the Abstract Syntax Tree (AST)
• explicit in Control Flow Graph (CFG), which captures the order of
execution and possible paths (assuming the targets of calls are known)

blocks may lead to which other blocks

• 0-CFA is a context-insensitive (or mono-variant) analysis
• It does not distinguish various instances of program variables and program
